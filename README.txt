A selenium scraper written in python to obtain data from the student timetable service of UTAS

INSTALLATION INSTRUCTIONS
1. Open powershell here (SHIFT+RIGHT CLICK in folder, OPEN POWERSHELL HERE)
2. python -m venv env
3. .\env\Scripts\activate
4. pip install -r "requirements.txt"
5. Run main in IDE

Must have chromedriver and place in the same folder as main
